The Secondaries command is used to get the list of secondary weapons.

## Usage

### Slash Command

- `/secondaries`
- `/secondaries pistol`
- `/secondaries launcher`

### Prefixed Command

- `!secondaries`
- `!secondaries pistol`
- `!secondaries launcher`

## Response

The response will display a list of all secondary weapon types in the game.

### Pistols

The response will display a list of all pistols in the game.

### Launchers

The response will display a list of all launchers in the game.
