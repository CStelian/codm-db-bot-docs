The maps command is used to get the list of maps.

## Usage

### Slash Command

- `/maps`
- `/maps <mp / br / zombies>`
- `/maps <map name>`

### Prefixed Command

- `!maps`
- `!maps <mp / br / zombies>`
- `!maps <map name>`

## Response

The response will display a list of all maps in the game.
