The scorestreaks command is used to get the list of scorestreaks.

## Usage

### Slash Command

- `/scorestreaks`
- `/scorestreaks <scorestreak name>`

### Prefixed Command

- `!scorestreaks`
- `!scorestreaks <scorestreak name>`

## Response

The response will display a list of all scorestreaks in the game.
