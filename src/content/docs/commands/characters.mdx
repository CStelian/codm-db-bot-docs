The characters command is used to get the list of characters.

## Usage

### Slash Command

- `/characters`
- `/characters <character type>`
- `/characters <character name>`

### Prefixed Command

- `!characters`
- `!characters <character type>`
- `!characters <character name>`

## Response

The response will display a list of all characters in the game.
