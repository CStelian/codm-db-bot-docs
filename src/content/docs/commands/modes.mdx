The modes command is used to get the list of game modes.

## Usage

### Slash Command

- `/modes`
- `/modes <mp / br / zombies>`
- `/modes <mode name>`

### Prefixed Command

- `!modes`
- `!modes <mp / br / zombies>`
- `!modes <mode name>`

## Response

The response will display a list of all game modes in the game.
