The classes command is used to get the list of operator classes available in Battle Royale.

## Usage

### Slash Command

- `/classes`
- `/classes <class name>`

### Prefixed Command

- `!classes`
- `!classes <class name>`

## Response

The response will display a list of all operator classes available in Battle Royale in the game.
