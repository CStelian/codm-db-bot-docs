The skills command is used to get the list of operator skills available in Multiplayer.

## Usage

### Slash Command

- `/skills`
- `/skills <skill name>`

### Prefixed Command

- `!skills`
- `!skills <skill name>`

## Response

The response will display a list of all operator skills available in Multiplayer in the game.
